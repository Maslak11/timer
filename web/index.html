Quick Access
How to connect:
1. Open the StageTimer desktop app on the operator PC
2. Click Output Links in the toolbar
3. Copy the IP address shown (or scan the QR code)
4. Enter the IP above and click Connect

All devices must be on the same Wi-Fi network.
🎮 Bitfocus Companion API
Add a Generic HTTP device in Companion pointing to:
http://[desktop-ip]:7001/api/status

Key endpoints:
POST /api/timer/start
POST /api/timer/pause
POST /api/timer/next
POST /api/blackout?state=true
POST /api/timer/adjust?seconds=60
